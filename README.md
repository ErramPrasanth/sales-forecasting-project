# Sales Performance Analysis & Forecasting

## Overview
This project analyzes historical sales data to understand trends, seasonality, and forecast future sales using machine learning.

## Dataset
Sample Superstore Dataset  
https://www.kaggle.com/datasets/vivek468/superstore-dataset-final

## Tools Used
- Python
- Pandas, NumPy
- Matplotlib
- Scikit-learn

## Steps Performed
- Data Cleaning
- Exploratory Data Analysis
- Feature Engineering
- Regression Model
- Model Evaluation

## Result
The model captures sales patterns and provides reasonable sales predictions.
